# Installation

1. Tapez `npm install`
2. Remplir les informations manquantes dans `src/config/default.js`

# Démarrer

Lancer le serveur :

`npm run dev`

Lancer localtunnel

`npm run lt`

# Créer un menu persistant

Lancer la commande (Linux) :

`TOKEN=<token de messenger> sh ./bin/persistent-menu.sh
